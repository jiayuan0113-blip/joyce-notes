# Reflection Frameworks

Use these frameworks as background. Do not force all of them into every user response.

Checked: 2026-06-21.

## 1. After Action Review

Best for: reviewing yesterday's goals or a finished event.

Core move:

1. What did I intend to accomplish?
2. What actually happened?
3. Why was there a gap between intention and execution?
4. What will I adapt, repeat, or stop next time?

How to use in this skill:

- Use AAR as the main skeleton for missed goals.
- Keep attention on facts, outcomes, and controllable execution.
- Ask "what should change next time?" before giving advice.

Source: Wharton Executive Education, "After-Action Reviews: A Simple Yet Powerful Tool" (2021), https://executiveeducation.wharton.upenn.edu/thought-leadership/wharton-at-work/2021/07/after-action-reviews-simple-tool/

## 2. Gibbs Reflective Cycle

Best for: a user who has strong emotions around an unfinished task.

Core move:

1. description
2. feelings
3. evaluation
4. analysis
5. conclusion
6. action plan

How to use in this skill:

- Use the feelings step to acknowledge emotion without letting it dominate.
- Use evaluation and analysis to separate "what happened" from "what it means."
- End with an action plan, not a self-judgment.

Source: University of Edinburgh Reflection Toolkit, "Gibbs' Reflective Cycle", https://reflection.ed.ac.uk/reflectors-toolkit/reflecting-on-experience/gibbs-reflective-cycle

## 3. Kolb Experiential Learning

Best for: explaining why repeated reflection creates compounding growth.

Core move:

1. experiencing
2. reflecting
3. thinking
4. acting

How to use in this skill:

- Treat each day as an experiment.
- Reflection turns experience into a hypothesis.
- Tomorrow's goal tests that hypothesis.

Source: Institute for Experiential Learning, "What Is Experiential Learning?", https://experientiallearninginstitute.org/what-is-experiential-learning/

## 4. WOOP

Best for: translating a goal into obstacle-aware planning.

Core move:

1. wish
2. outcome
3. obstacle
4. plan

How to use in this skill:

- After choosing today's goal, identify the most likely obstacle.
- Turn it into a simple plan instead of relying on willpower.

Source: WOOP my life, "What is WOOP?", https://woopmylife.org/

## 5. Implementation Intentions

Best for: making the next action concrete.

Core move:

```text
If situation Y happens, then I will do behavior Z.
```

How to use in this skill:

- Every high-risk goal should have one if-then plan.
- Avoid vague plans such as "exercise more" or "focus better."
- Use a specific cue and specific response.

Source: National Cancer Institute DCCPS, "Implementation Intentions", https://cancercontrol.cancer.gov/brp/research/constructs/implementation-intentions

## 6. SMART Objectives

Best for: checking whether a goal is concrete enough.

Core move:

- specific
- measurable
- achievable
- relevant
- time-bound

How to use in this skill:

- Do not overcomplicate daily goals with corporate goal language.
- Use SMART only as a quality check: can the user verify completion tomorrow?

Source: SAMHSA, "Setting Goals and Developing Specific, Measurable, Achievable, Relevant, and Time-bound Objectives", https://www.samhsa.gov/sites/default/files/nc-smart-goals-fact-sheet.pdf

## 7. Self-Compassion

Best for: preventing reflection from becoming self-attack.

Core move:

- Self-compassion is not avoiding responsibility.
- It creates enough psychological safety to acknowledge mistakes and repair them.

How to use in this skill:

- Replace "I failed again" with "what made completion harder this time?"
- Keep accountability, remove humiliation.

Source: self-compassion.org, "Exploring the Meaning of Self-Compassion and Its Importance", https://self-compassion.org/what-is-self-compassion/
