# Skills

这里放可以直接复制、下载、复用的 AI Skill。

## Skill 列表

- [skill-curator](skill-curator/SKILL.md) — 盘点重复与触发冲突，安全地重写、禁用、归档或清理 AI Skills。
- [daily-reflection-loop](daily-reflection-loop/SKILL.md) — 每日 1-3 目标复盘、归因、可控动作和明日计划。

## 怎么下载 / 使用

### Codex / 支持 Skill 文件夹的工具

下载对应的整个目录，例如：

```text
skills/skill-curator/
skills/daily-reflection-loop/
```

`skill-curator` 目录里包含：

- [SKILL.md](skill-curator/SKILL.md)
- [agents/openai.yaml](skill-curator/agents/openai.yaml)
- [scripts/skill_inventory.py](skill-curator/scripts/skill_inventory.py)
- [references/description-guide.md](skill-curator/references/description-guide.md)
- [references/safety-policy.md](skill-curator/references/safety-policy.md)

`daily-reflection-loop` 目录里包含：

- [SKILL.md](daily-reflection-loop/SKILL.md)
- [agents/openai.yaml](daily-reflection-loop/agents/openai.yaml)
- [references/reflection-frameworks.md](daily-reflection-loop/references/reflection-frameworks.md)

### 手机端 / 普通 AI 工具

如果你的 AI 工具不支持 Skill 文件夹，直接打开这个单文件版，复制全文给 AI：

- [skill-curator.skill.md](skill-curator.skill.md)
- [daily-reflection-loop.skill.md](daily-reflection-loop.skill.md)
